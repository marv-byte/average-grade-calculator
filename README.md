# Average Grade Calculator

This tool allows you to calculate your average grades. It is designed for multiple calculations and saves the history to compare results.

## Features

* **Flexible subject count:** You can choose how many subjects you want calculated.
* **Universal grade system:** Works with every number-based grading system (e.g. 1-6 or 0-15).
* **Storage:** You can save your average with a custom name in the storage, if you want.
* **Review:** You can see the history by typing `end` in the main prompt.
* **Error handling:** To prevent crashes, it uses `try-except`. When a wrong input is given, it just jumps back to the main prompt.

**Example run:** Amount of subjects: `3` and then add the grades `1` , `3` ,  `5`

## Future improvements
* **Exporting the storage:** Saving it permanently in a `.txt` file, so it doesn't get lost.
